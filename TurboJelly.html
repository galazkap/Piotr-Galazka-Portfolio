Turbo Jelly

3D platform game, created as a university project. I was mainly inspired by games like "Rayman 2" and "Banjo-Kazooie". The game features the basic mechanics found in these types of games, such as fighting enemies, climbing and solving puzzles. I am most satisfied with the implementation of the main character's voice in this game.

The 3D objects, textures and sounds used come from the Unity Asset Store.
