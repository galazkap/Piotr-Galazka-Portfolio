Red Ridding Hood

An alternative version of the famous story of Red Riding Hood. A 2D action platform game, where the goal is to go through the forest and reach grandma's house. The main mechanic of the game is to use an oil lamp (light) to see anything that is not normally visible.

My main task in this project was to design and implement game mechanics.
